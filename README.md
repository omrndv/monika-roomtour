# 👑 Monika Roomtour Management

Sistem manajemen reservasi "Room Tournament" Mobile Legends berbasis Laravel. Aplikasi ini dirancang untuk memantau omset, mengelola jadwal booking anti-tabrakan, dan melacak bagi hasil staff secara real-time.

### 🚀 Fitur Utama
* **Dashboard Operasional**: Pantau omset dan total order harian.
* **Anti-Double Booking**: Sistem validasi jadwal agar tidak ada jam yang tumpang tindih.
* **Weekly Grid View**: Visualisasi jadwal mingguan ala tabel untuk cek slot kosong.
* **Staff Tracking**: Kelola status gaji dan bagi hasil antara Staff.
